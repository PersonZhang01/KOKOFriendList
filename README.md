# KOKOFriendList
iOS Swift Project  
使用 Combine framework 實作 MVVM 架構的程式  
取得KOKO好友列表  

- 點擊右上方Scan可以切換4種好友列表
- 好友列表支援下拉更新
- 點擊搜尋框，好友列表畫面上推置頂，且下拉更新或點擊Scan可復原
